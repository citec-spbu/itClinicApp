# CiteC

## RU

Kotlin Multiplatform mobile client для Android и iOS.

CiteC помогает участникам IT-проектов просматривать проекты, отслеживать рейтинг и статистику по репозиторной активности, а также управлять профилем и обратной связью в одном мобильном клиенте.

- Основные разделы: проекты, рейтинг, статистика, профиль, настройки, обратная связь.
- Backend-репозиторий: [Registry](https://github.com/AngW1SH/Registry)
- Документация: `website/`
- Публикация docs-site: GitHub Pages через Actions при push в `main`

## EN

Kotlin Multiplatform mobile client for Android and iOS.

CiteC helps IT project participants browse projects, track ranking and repository-based statistics, and manage profile and feedback flows from a single mobile client.

- Main areas: projects, ranking, statistics, profile, settings, feedback.
- Backend repository: [Registry](https://github.com/AngW1SH/Registry)
- Documentation: `website/`
- Docs deployment: GitHub Pages via Actions on push to `main`
