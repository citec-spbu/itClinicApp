Mobile CD Artifact

itClinicApp APK Delivery

This container publishes the signed Android release build as a downloadable artifact. It is intended for mobile delivery and CI/CD verification, not for backend deployment.

Download APK Open CI/CD Documentation

What this image contains

  • the latest signed Android release APK
  • the version-specific Android update manifest uploaded to GitHub Releases
  • the CI/CD documentation page
  • a lightweight nginx server for file delivery

How to use it

  • open the container root page in a browser
  • download the APK from the provided link
  • share the image tag as a reproducible mobile build artifact
