itClinicApp

Docker-образ публикует статический showcase мобильного приложения и его документацию. Это app-only артефакт: без backend-контейнера и без backend-зависимостей.

README Project Statistics Personal Statistics
CI

Что проверяется

  • lintDebug для Android/Compose static analysis
  • testDebugUnitTest для unit tests модуля composeApp
  • артефакты lint и test report сохраняются в GitLab job artifacts
CD

Что публикуется

  • образ app-showcase в GitLab Container Registry
  • теги: latest, $CI_COMMIT_SHORT_SHA, $CI_COMMIT_REF_SLUG или git tag
  • внутри контейнера запущен nginx со статикой проекта
