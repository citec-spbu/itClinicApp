# Project Statistics Screens

## Purpose

Project statistics explain repository activity and derived quality metrics for a selected project.

## Design

- summary score and project context;
- metric blocks for commits, issues, pull requests, churn, ownership, and similar categories;
- detail navigation for deeper metric inspection.

## Requests

- project statistics loading for the active period and project;
- export-related actions where enabled;
- detail metric loading when a dedicated screen requires it.
