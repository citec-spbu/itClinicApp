# Authentication

## Access modes

The client supports two entry paths:

- authenticated mode through GitHub-based authorization;
- guest mode with restricted access.

## Guest restrictions

Guest users can browse only the public part of project data. Ranking, statistics, and private project sections remain unavailable.

## Auth-dependent behavior

Authentication changes:

- tab availability;
- visibility of project detail blocks;
- profile data loading;
- analytics identity and session lifecycle.
