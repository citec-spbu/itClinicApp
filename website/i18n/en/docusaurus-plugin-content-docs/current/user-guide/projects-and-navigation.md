# Projects and Navigation

## Main sections

The app is organized around three main tabs:

- Projects;
- Ranking / Statistics;
- Information.

## Typical flow

Users start from onboarding or the main shell, open the projects list, inspect a project, and then move into ranking, statistics, or personal settings depending on access level.

## Navigation model

The tab shell is stable. Detail screens, filters, and settings flows open above it without resetting the currently selected tab.
