# Product Summary

## What CiteC does

CiteC is a Kotlin Multiplatform mobile client for Android and iOS. It gives users a single place to:

- browse projects and inspect project details;
- review project and student rankings;
- work with personal and project statistics;
- manage profile and app settings;
- send feedback from inside the app.

## Main user groups

- students;
- mentors and supervisors;
- customers and external stakeholders;
- guest users with restricted access.

## Product focus

The client is centered around repository-driven activity data, ranking, and statistics rather than task execution inside the app itself.
