# Product Summary

`CiteC` — Kotlin Multiplatform мобильный клиент для IT-клиники СПбГУ.

## Что умеет приложение

- показывать список проектов и детальные карточки проектов;
- поддерживать вход через GitHub и гостевой режим;
- показывать рейтинг проектов и студентов по метрикам активности;
- показывать проектную и личную статистику;
- экспортировать статистику;
- редактировать профиль и базовые настройки;
- отправлять обратную связь.

## Для кого приложение

- студенты;
- руководители и наставники;
- заказчики;
- гости без авторизации.

- `composeApp/src/androidMain/kotlin/**`
