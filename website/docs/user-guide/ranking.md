# Ranking

## Что есть на экране ranking

Клиент поддерживает ranking для:

- проектов;
- студентов.

## Что можно настраивать

На клиенте уже есть:

- выбор метрик;
- диапазон дат;
- порог для rapid PR;
- клиентские filter presets в рамках текущей логики экрана.

## Важный нюанс

Ranking не опирается на один готовый backend endpoint со всеми параметрами. Часть ranking logic клиент пересчитывает локально по metric snapshot-данным.

Из этого следуют ограничения:

- возможны client-side эвристики;
- часть score считается как workaround;
- docs по ranking надо читать вместе с разделом `Backend Gaps`.
